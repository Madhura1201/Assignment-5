# Project Title

DATA1200 Assignment 5 – Data Analysis using Python

# Short Description

This project contains a Jupyter Notebook that performs data analysis using Python. It focuses on exploring datasets, calculating statistical measures, and visualizing results using graphs and plots. The assignment demonstrates how data can be processed and interpreted effectively.

# Getting Started

To get started, download or clone the repository and open the notebook file in Jupyter Notebook or Google Colab. Run the cells step by step to view outputs and results.

# Prerequisites

Python 3.x
Jupyter Notebook or Google Colab
Pandas
NumPy
Matplotlib

# Installing

Install the required libraries using:
pip install pandas numpy matplotlib

# Running the Tests

Run all cells sequentially
Observe outputs including tables, calculations, and graphs
